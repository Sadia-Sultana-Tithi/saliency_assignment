##  Installation (GazeGAN Model)

**Prerequisites**:
   - Python 3.8+
   - PyTorch 1.8+
   - CUDA (optional for GPU acceleration)

1. **Install dependencies**:
```bash
   pip install torch torchvision numpy opencv-python scipy tqdm 
```
3. **Training the Model**:
```bash
   python train.py
```
3. **Evaluating the Model**:
```bash
    python evaluate.py
